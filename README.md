# Détection SQL/XSS sur API Web avec IA

Ce projet vise à détecter les injections SQL et XSS dans des requêtes/params d'API en utilisant du machine learning.

## Contenu du repo
- `scripts/` : scripts pour valider, nettoyer/split et entraîner un modèle.
- `data/` : dossier attendu pour le dataset.
- `tests/` : tests unitaires légers.
